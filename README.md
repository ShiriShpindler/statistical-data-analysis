# Statistical Data Analysis

Statistical analysis and biological mutation data processing using Python.

This project includes exploratory data analysis, mutation analysis, statistical inference, and hypothesis testing.

---

## Technologies
- Python
- Pandas
- NumPy
- SciPy
- Statistics

---

## Project Structure

### genomic_mutation_analysis.py
Mutation dataset analysis:
- Mutation counting
- Tissue-specific mutation analysis
- Sample filtering
- Exploratory data analysis
- Grouped statistical summaries

### statistical_hypothesis_testing.py
Statistical inference and probability analysis:
- Maximum likelihood estimation (MLE)
- Hypothesis testing
- P-value computation
- Exponential distribution analysis
